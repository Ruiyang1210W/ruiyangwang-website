# Ruiyang Wang's Portfolio

A modern, bilingual personal portfolio website showcasing technical projects and creative artwork. Built with Next.js and featuring a seamless blend of software engineering excellence and artistic vision.

## Features

- **Bilingual Support**: Full internationalization with English and Chinese translations using i18next
- **Technical Portfolio**: Showcasing software engineering projects including AI/ML, web development, and mobile applications
- **Creative Gallery**: Digital art, traditional artwork, logo design, and character portfolios
- **Character Showcase**: Detailed character profiles with bios and traits for original characters
- **Responsive Design**: Fully responsive UI optimized for all devices
- **Modern UI/UX**: Clean, gradient-rich design with smooth animations and transitions
- **Contact Integration**: Built-in contact form for collaboration opportunities

## Tech Stack

- **Framework**: [Next.js 15](https://nextjs.org/) (App Router)
- **Frontend**: [React 19](https://react.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Internationalization**: [i18next](https://www.i18next.com/) / [react-i18next](https://react.i18next.com/)

## Getting Started

### Prerequisites

- Node.js 18.x or higher
- npm, yarn, pnpm, or bun

### Installation

1. Clone the repository:
```bash
git clone https://github.com/YOUR_USERNAME/ruiyang-portfolio.git
cd ruiyang-portfolio
```

2. Install dependencies:
```bash
npm install
# or
yarn install
# or
pnpm install
```

3. Run the development server:
```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.
